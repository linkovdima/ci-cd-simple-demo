# Демонстрация CI/CD Пайплайна

Это простой проект для демонстрации работы CI/CD пайплайна с GitHub Actions.

## Что делает этот пайплайн:

1. **Автоматическая проверка** - при каждом push проверяет наличие необходимых файлов
2. **Создание артефакта** - упаковывает файлы для развертывания
3. **Автоматическое развертывание** - публикует сайт на GitHub Pages

## Файлы проекта:
- `index.html` - главная страница сайта
- `.github/workflows/deploy.yml` - конфигурация CI/CD пайплайна

## Как запустить:
1. Сделайте push в репозиторий
2. Смотрите результат в Actions -> Pages
